## What is VCVQ?

VCVQ is a real-time multiplayer quiz game designed specifically for car trips. It uses AI to generate quiz questions on any topic and supports 2-5 players with an intuitive drag-and-drop or click interface.

### Key Features

✅ **Car-Friendly Design** - Perfect for road trips  
✅ **AI-Generated Content** - Questions, topics, and player names  
✅ **Bilingual** - Swedish and English support  
✅ **Tesla Compatible** - Optimized for Tesla Model Y browser  
✅ **Secure** - Comprehensive security features  
✅ **Open Source** - MIT License
